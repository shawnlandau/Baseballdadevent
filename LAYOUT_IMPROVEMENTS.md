## 🎨 DESIGN PHILOSOPHY

### Visual Hierarchy
1. **Bold, Commanding Titles** - Immediate attention grabbers
2. **Clear Subsections** - Easy navigation through content
3. **Ample White Space** - Prevents overwhelming users
4. **Consistent Typography** - Montserrat for impact, Open Sans for readability

### Color Usage
- **Primary Red (#c8102e)**: Action items, accents, borders
- **Navy (#0a1628)**: Backgrounds, grounding elements
- **Gold (#FFD700)**: Highlights, countdown, special emphasis
- **Gradients**: Smooth transitions between colors

### User Experience
- **Smooth Animations**: All transitions at 0.3s ease
- **Hover Feedback**: Cards lift, buttons respond
- **Clear CTAs**: Large, obvious action buttons
- **Mobile-First**: Fully responsive on all devices

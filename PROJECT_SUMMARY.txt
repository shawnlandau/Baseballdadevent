HOW TO DEPLOY
================================================================================

METHOD 1 - DRAG & DROP (2 minutes):
1. Go to https://app.netlify.com/drop
2. Drag the /home/user/webapp folder onto the page
3. Done! Site is live with a Netlify URL

METHOD 2 - GITHUB + NETLIFY (Recommended):
1. Create GitHub repo
2. Push code: git push -u origin main
3. Connect to Netlify dashboard
4. Auto-deploy on every push

See DEPLOY.md for detailed instructions.

- Update DNS at registrar (see README.md for details)

- Netlify Dashboard → Forms → dads-registration
- Export as CSV or set up email notifications
